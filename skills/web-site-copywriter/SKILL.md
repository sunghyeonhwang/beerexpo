# Web-Site-Copywriter

## When to use
- 웹사이트 페이지 카피를 배포 가능 수준으로 다듬어야 할 때
- 원문이 장황/중복되어 핵심 메시지 재구성이 필요할 때
- 연도 혼재(예: 2024/2025/2026)로 최신성 정리가 필요할 때

## Output format (required)
1. Hero: 한 줄 가치 제안
2. 핵심 요약: 3개 불릿
3. 참석자별 안내:
- 참가사 관점
- 바이어 관점
- 참관객 관점
4. 프로그램/근거 데이터
5. CTA (1차/2차/3차)
6. SEO Draft (Title/Description/Keywords)
7. 검수 메모 (최신성/아카이브/운영 확인)

## Copy rules
- 첫 화면은 짧게, 하단은 충분히 상세하게 (요약 + 상세 2단 구조)
- 과장 표현보다 실행 정보 우선 (일정, 장소, 절차, 조건, 문의)
- 숫자 데이터는 출처 연도를 명시 (`전년도 기준` 등)
- 같은 의미 반복 문장을 제거
- 문장 길이는 짧게, 모바일 스캔 가능하게 구성
- **모바일 가독성:** Hero 섹션은 2줄을 넘지 않으며, 한 문장은 최대 40자를 초과하지 않음.

## Freshness rules
- 현재 연도/행사 연도와 불일치하는 운영 정보는 본문 노출 금지
- 과거 실적 데이터는 허용하되 `전년도 기준` 라벨 필수
- 마감일/비용/장소는 최종 검수 메모에 재확인 항목으로 남김

## Finalization checklist
- 페이지 목적이 1문장으로 설명되는가
- 타겟(참가사/바이어/참관객)별 행동 경로가 보이는가
- CTA가 실제 라우트와 연결 가능한가
- SEO 문구가 페이지 주제와 일치하는가
- 과거 연도 문구가 오해 없이 처리됐는가
